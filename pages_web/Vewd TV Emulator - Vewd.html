Vewd TV Emulator

Vewd Emulator allows you to test and debug apps, as it contains a full implementation of Vewd Core and the Vewd Media Player Module that can run on your own PC. The Emulator helps verify that your content works on Vewd-certified devices, and that it’s ready for Vewd Certification.

We offer three types of emulators to choose from; two of which are Chromium-based, and the third is based on Presto. Read more below for a short explanation of each emulator.

Chromium based
Vewd TV Emulator
(Standalone)

Download the Emulator

This emulator environment runs on top of Oracle VirtualBox and is suitable for developers performing extensive, in-depth tests of web applications based on Chromium or Vewd Core 4.X versions. This package comes closest to emulating an actual embedded system running Vewd, with exactly the same browser engine versions and closely matching platform media playback capabilities. The emulator comes packaged as an Oracle VirtualBox image and can be run on Windows, Mac and Linux machines.

Chromium based
Vewd TV Simulator
(Browser Extension)

Download the Simulator

The simulator runs directly in any Chromium based browser (such as Chrome or Opera) as an extension, and is the quickest way to test web applications built using HTML5. This is the recommended tool for early development, however we recommend using the standalone emulator when approaching the final development phase.

For more about how to use the Vewd TV Emulator, as well as articles detailing how to develop content for our TVs, visit the Vewd Developer Community.
