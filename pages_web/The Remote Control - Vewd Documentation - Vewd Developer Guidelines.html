The Remote Control

Aller directement à la fin des métadonnées
  • Créée par Kevan Qureshi le déc. 17, 2016
Aller au début des métadonnées
Overview

The Remote Control is a fixed part of the Control Panel. It appears on the right side of the screen and allows you to control the Emulator input from the Control Panel. It should be used to test how your web app behaves under a simulated TV input environment.


The Control Panel Remote Control

 

The Remote Control can be roughly divided into 3 sections:

  • Vewd TV Emulator Buttons: These buttons affect the Emulator as a whole. For example, you can restart the Emulator or shut it down completely.
  • Vewd Core Buttons: These buttons affect the Vewd Core only. For example, you can navigate back and forth between pages in the Emulator, reload the current url, etc.
  • Buttons available to Web Apps: The rest of the buttons are available to your web app. These include the up, down, back, etc keys.

Remote Control Profiles

Overview

The Remote Control comes with some selectable profiles. Each profile has its own set of keys that are enabled and disabled. Even within the same set of keys, the key mappings may be different.

  • ACP Compliant: The key mappings correspond to those defined in the Vewd Application Certification Program (http://acp.otvs.tv). All keys are supported, both labeled 'MANDATORY' and 'CONDITIONALLY REQUIRED'.
  • OTVS Legacy: The key mappings correspond to Legacy Vewd App Store (OTVS) applications.
  • ACP Restricted: The key mappings correspond to the Application Certification Program (ACP) like in the first option. However, the difference is that only those keys marked as 'MANDATORY' in the ACP spec are allowed. All other keys are grayed out.

The key mappings for these different profiles are now described.
